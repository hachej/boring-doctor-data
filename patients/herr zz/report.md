CAT :
Douleur thoracique = urgence jusqu'à preuve du contraire. Éliminer 6 diagnostics (4 UV : PIED) : péricardite, SCA, embolie pulmonaire, dissection aortique, pneumothorax compressif, rupture œsophage.
Règle ABCDE initiale. Bilan stéréotypé immédiat : constantes (TA 2 bras, SpO2, fR, fC, GSC), ECG 18D (dans les 10min), troponine US (T0, puis T1h si Tn-us, ou T3-6h si Tn standard), radio thorax F. 
Si signes de gravité (détresse resp/hémodynamique/neuro) → USIC/réanimation immédiate. Transfert centre coronarographie H24.
Rechercher FdRCV (âge, sexe, ACTDf <55H/<65F, tabac actif ou sevré <3a, diabète, HTA, dyslipidémie, IRC), ATCD coronarien/MTEV/cancer, terrain Marfan/connectivite, prise cocaïne (coronarospasme).
Caractériser douleur : irradiation (bras/mâchoire = SCA, dos = dissection aortique, basi-thoracique pleurétique = EP), type, facteurs soulageant (trinitrine <3min = coronaire, antéflexion = péricardite).
Si ST+ convexe systématisé ou BBG aigu : SCA ST+ → coronarographie <2h + bi-AAP (aspirine 250mg IV + P2Y12 : prasugrel 60mg PO ou ticagrelor 180mg PO) + HNF/HBPM. CI : inhibiteur phosphodiestérase 5 <48h. Objectif FMT <120min douleur-ballon. Puis USIC, ETT post-reperfusion.
Si ST- avec Tn+ : SCA ST- → stratification risque : si très haut risque (instabilité, douleur réfractaire, TV, élévation Tn massive) → coronarographie <2h ; si haut risque (GRACE >140, Tn+, ST-, diabète, IRC) → coronarographie <24h.
Si suspicion dissection aortique (douleur déchirante syncopale migratrice, asymétrie TA >20mmHg, abolition pouls, souffle IA, déficit neuro) : score ADD-RS (≥2 = haute proba) → TDM thoracique injecté urgence (ou ETO). CI absolue anticoagulation + AAP. Urgence hypertensive : β-bloquant IV (si pas de choc) ou IC IV (si choc) pour TA 120/80mmHg. Type A (ascendante) → chirurgie urgente.
Si EP suspectée (dyspnée + polypnée, signes TVP, douleur basi-thoracique, facteurs risque MTEV) : score Wells/Genève → si risque faible/intermédiaire : D-dimères (si >500µg/L ou >âge×10) → angio-TDM thoracique ; si risque fort → angio-TDM direct. ECG : tachycardie sinusale, S1Q3, BBD, T- V1-V3.
Si péricardite suspectée (contexte viral/fébrile, douleur améliorée antéflexion/aggravée inspiration/décubitus) : ECG (ST+ concave diffus sans miroir, sous-PQ), frottement péricardique, CRP/troponines, ETT (recherche épanchement/tamponnade). Si tamponnade (pouls paradoxal, TJ, signes choc) → drainage péricardique urgence.
